# Deal Radar Web

Next.js App Router demo for the Deal Radar cache-first deal intelligence flow.

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```

Local web target:

```txt
http://localhost:13000
```

Backend defaults:

```env
API_BASE_URL=http://localhost:18000
NEXT_PUBLIC_API_BASE_URL=http://localhost:18000
```
